# Data Structure

A **data structure** is a specialized format for organizing, processing, retrieving, and storing data in a computer's memory so that it can be accessed and modified efficiently.


Data structures are broadly divided into two major categories:

- Linear data structures

- Non-linear data structures


## Linear Data structures

A **linear data structure** is a storage structure where data elements are arranged sequentially or linearly, meaning each element is attached to its unique previous and next adjacent elements. Because elements are arranged in a single level, they can be fully traversed in a single run. This makes them highly predictable and straightforward to implement compared to hierarchal non-linear structures.
